# AFS LifeApps Assets

Este repositório é utilizado exclusivamente para hospedar os ativos visuais (imagens, logos e ícones) consumidos pelos aplicativos e serviços da marca **AFS LifeApps**.

## 🛡️ Propósito
O acesso público a este repositório é mantido estritamente para garantir o funcionamento técnico de integrações externas e gateways de pagamento (como PayPal e Revolut) que exigem URLs públicas para exibição de branding durante transações.
